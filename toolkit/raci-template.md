# RACI Template for AI-Assisted Decision Rights

Standard RACI (Responsible, Accountable, Consulted, Informed), adapted for AI-assisted workflows — with an added row type for "AI role" so the template makes explicit what the model is and isn't doing.

## Template

| Workflow step | AI role | Responsible (does the work) | Accountable (owns outcome) | Consulted (input before) | Informed (told after) |
|---|---|---|---|---|---|
| Research / background gathering | Drafts/summarizes | | | | |
| First draft | Generates | | | | |
| Editorial review | None (human only) | | | | |
| Fact-check / compliance check | None (human only) | | | | |
| Final approval | None (human only) | | | | |
| Publication | Assists (scheduling, formatting) | | | | |
| Post-publication monitoring | Assists (analytics, sentiment) | | | | |

## Filled example — internal newsletter (Tier 1)

| Workflow step | AI role | Responsible | Accountable | Consulted | Informed |
|---|---|---|---|---|---|
| Research / background gathering | Drafts/summarizes | AI + comms associate | Comms manager | — | — |
| First draft | Generates | AI | Comms associate | — | — |
| Editorial review | None (human only) | Comms associate | Comms manager | — | — |
| Fact-check / compliance check | None (human only) | Comms associate | Comms manager | HR (if policy-related) | — |
| Final approval | None (human only) | Comms manager | Comms manager | — | Team |
| Publication | Assists | Comms associate | Comms manager | — | — |
| Post-publication monitoring | Assists | Comms associate | Comms manager | — | Comms manager |

## Filled example — crisis statement (Tier 4)

| Workflow step | AI role | Responsible | Accountable | Consulted | Informed |
|---|---|---|---|---|---|
| Research / background gathering | Drafts/summarizes | AI + comms lead | CCO | Legal | — |
| First draft | Generates (structure only) | Comms lead (full rewrite) | CCO | Legal, CEO | — |
| Editorial review | None (human only) | Comms lead | CCO | Legal, CEO | — |
| Fact-check / compliance check | None (human only) | Legal | CCO | — | — |
| Final approval | None (human only) | CCO | CEO/CCO | Legal | Board (if material) |
| Publication | Assists (distribution only) | Comms lead | CCO | — | — |
| Post-publication monitoring | Assists | Comms team | CCO | — | CEO |

---

**Principle:** the "AI role" column should never say "decides" or "approves." AI drafts, assists, and generates — accountability and approval stay with named humans, always.
