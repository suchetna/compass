# Compass
### An Operating Model Framework for AI-Native Corporate Communications

Most communications functions have adopted AI tools. Very few have redesigned how the function *operates* around them. That gap — not tool access — is what separates comms teams that show up in the C-suite with evidence of impact from teams still explaining why the pilot didn't scale.

BCG's 2026 research on corporate communications found that nearly 70% of Chief Communications Officers describe their function as an AI laggard, and the single largest obstacle isn't budget or tooling — it's the absence of operating-model design capability. Compass is built to close exactly that gap: a practical, adaptable framework for redesigning the comms operating model — people, process, technology, and governance — around AI, rather than bolting AI onto an unchanged model.

This is not a tool-selection guide. It's a methodology for the harder, less glamorous work: deciding what changes about roles, workflows, decision rights, and risk controls when AI moves from a sidebar experiment to infrastructure.

---

## Why this exists

Every comms leader today has run a pilot. Fewer have answered:
- Which decisions should still require human sign-off, and why — not as a blanket policy, but by workflow
- How team roles and career paths change when drafting, monitoring, and synthesis are partially automated
- What governance looks like when the function's output volume triples but headcount doesn't
- How to measure whether AI adoption is actually creating value, versus just activity

Compass gives you a structured way to answer these, then a set of working documents to run the redesign.

---

## What's inside

**`/framework`** — the methodology, in four parts:
1. **Maturity Model** — where a comms function sits today, in blunt operational terms
2. **Operating Model Pillars** — the four dimensions that actually need redesigning
3. **Assessment Methodology** — how to diagnose your current state honestly
4. **Implementation Roadmap** — a phased path from laggard to leader

**`/toolkit`** — working documents to run the redesign in a real organization:
- AI readiness assessment (scorable)
- Workflow audit checklist
- Governance and risk matrix
- RACI template for AI-assisted decision rights
- ROI measurement template

**`/diagrams`** — visual model of the operating system (Mermaid source)

**`/examples`** — an illustrative case study applying the full framework to a hypothetical rollout scenario, end to end (readiness score → workflow audit → risk tier → RACI → ROI). Clearly marked as illustrative, not an account of a real engagement — included to show the framework in motion.

---

## Who this is for

Comms leaders, consultants, and practitioners who need a working structure for an AI operating model conversation — not another trend report. Adapt freely; the value is in the scaffolding, not rigid prescription.
